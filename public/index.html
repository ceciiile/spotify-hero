TC-Hero

Download and build

$ git clone https://github.com/gforien/TC-Hero.git
$ cd TC-Hero
$ virtualenv launch_dir # optional
$ source launch_dir/activate # optional
$ pip install nodeenv # optional
$ nodeenv -p # optional
$ npm install
$ npm start
