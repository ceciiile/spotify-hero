TC-Hero

Download and build

$ git clone https://github.com/gforien/TC-Hero.git
$ cd TC-Hero
$ virtualenv launch_dir
$ source launch_dir/bin/activate
$ pip install nodeenv
$ nodeenv -p
$ npm install
$ # add secu.json to folder
$ npm start
